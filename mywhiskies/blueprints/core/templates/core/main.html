The Story

It started with too many bottles.

My Whiskies Online is a venture started by a whiskey nerd who also happens to be a data nerd. After accumulating more bottles than he could readily keep track of, it became evident that the solution was a database.

So the database was created. And then a web front-end. And things were good — but it only ran locally on a laptop, and could only be used by one person.

The whiskey community is all about sharing.

With that in mind, an effort was underway to build an application that anybody could use online.

This is that application.
